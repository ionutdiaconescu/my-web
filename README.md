# Ionut Diaconescu — Portfolio

Personal portfolio showcasing selected projects, skills, and contact options.

**Live:** https://ionutdiaconescu.com

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React Icons

## Features

- Sticky profile card with contact links
- Skills and overview sections
- Project gallery with preview modal and mobile swipe
- SEO-friendly metadata

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Project Structure

- app/components — UI sections (Profile, Overview, Skills, Work)
- app/data — content and data models
- public/images — local assets

## Deployment

Deployed on Vercel. Configure your domain in your hosting provider and update metadata in app/layout.tsx if needed.
